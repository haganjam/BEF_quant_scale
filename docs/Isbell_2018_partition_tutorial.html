How to calculate biodiversity effects across space and time (sensu Isbell et al.Â 2018, Ecology Letters)

Introduction

The field of biodiversity-ecosystem functioning (BEF) studies the relationship between biological diversity and different ecosystem processes or functions (primary productivity, nutrient cycling etc.), (reviewed in Cardinale et al. 2012). One of the key tools in the BEF researcherâs toolbox is the partition proposed by Loreau and Hector (2001, Nature). This partition allows researchers to decompose a net biodiversity effect (i.e.Â the difference in mixture functioning relative to the expected functioning based only on monoculture yields and initial relative proportions or, relative expected yields, RYe) into two effects:

    1. The complementarity effect which describes processes related to facilitation and niche partitioning between species. This allows all species, on average, to perform better in mixture than expected based on their functioning in monoculture.
    1. The selection effect which describes the effect of biodiversity that comes from high functioning, dominant species being more likely to be present in a given community.

This partition has been very useful in the BEF field for helping to identify mechanisms driving biodiversity effects. However, this method generally only looks at the effect of biodiversity in single patches. It does not account for biodiversity effects that can arise at larger scales of space and time. For example, a single species may dominate a patch (i.e.Â strong selection effect) but if different species dominate different patches, we can get a complementarity effect arising at larger spatial scales.

Isbell et al.âs (2018, Ecology Letters) extension

Given the scale limitations of Loreau and Hectorâs (2001, Nature), Isbell et al.Â (2018, Ecology Letters) developed an extension of this method that calculates a net biodiversity effect across many times and places. This large-scale net biodiversity effect is then partitioned into a number of different effects that describe how biodiversity can effect ecosystem functioning at different scales of space and time. In Isbell et al.âs (2018, Ecology Letters) partition, the net biodiversity effect (NBE) is partitioned into:

    1. Total complementarity (TS)
    1. Total selection (TS)

TS is then further partitioned into two different effects

    1. Non-random overyielding (NO)
    1. Total insurance (IT)

Finally, IT is then partitioned into four different effects

    1. Average selection (AS)
    1. Spatial insurance (SI)
    1. Temporal insurance (TI)
    1. Spatio-temporal insurance (ST

For descriptions of these different effects, see Isbell et al.Â (2018, Ecology Letters) or Hagan et al.Â (in prep.).

Currently, general code for calculating these effects has not been developed (Isbell et al.Â 2018 included code for calculating these effects in specific cases). Therefore, we provide a script that contains a function that can calculate these effects for any number of species across any number of times and places:

  • 01_isbell_2018_partition.R

In addition, the script includes an extension proposed by Hagan et al.Â (in prep.) to incorporate uncertainty in expected relative yields (RYe) using samples from a Dirichlet distribution. This is a tutorial that explains how to use this script.

Currently, the script is hosted on Github: https://github.com/haganjam/BEF_quant_scale/blob/main/scripts/01_partition_functions/01_isbell_2018_partition.R

The main function used to calculate the biodiversity effects is:

# Isbell_2018_sampler(data, RYe_post = FALSE, N = 100, RYe, alpha_par = 4)

There are five arguments in the function. The argument data is the dataset on which to calculate the biodiversity effects. This data.frame must have five columns:

    1. sample - unique sample id for each place x time combination
    1. place - place from which data were collected (i.e.Â a code)
    1. time - time at which data were collected (i.e.Â a code)
    1. species - species name or code
    1. M - monoculture yield of the species at the specific time x place combination
    1. Y - mixture yield of the species at the specific time x place combination

Importantly, the time variable is simply a different time not an actual date. Therefore, a typical time-variable might be 1, 2 and 3 representing three different time points. In addition, all speciesâ names need to be present at all times and places in the data even if their monoculture or mixture functioning is zero.

Here is an example of a dataset that would be accepted by isbell_2018_sampler() which comes from Table 2 in Isbell et al.Â (2018, Ecology Letters).

The next argument, RYe_post, is a TRUE or FALSE argument. If TRUE, the function will calculate a distribution of each biodiversity effect where each different value assumes a different set of RYe values from a Dirichlet distribution. The number of samples to draw from the Dirichlet distribution is given by the N argument and the argument alpha_par is the alpha parameter in the Dirichlet distribution which determines the skewness of the distribution. An example of a set of 100 samples from the Dirichlet distribution with an alpha value of four for three species is given below. Each line is one sample and the numbers sum to one.

# load the relevant packages
require(ggplot2)
## Loading required package: ggplot2
require(gtools)
require(tidyr)
## Loading required package: tidyr
require(dplyr)

# draw samples from the Dirichlet distribution
species <- 3
p1 <- 
  lapply(1:100, function(x) {
  
  x <- gtools::rdirichlet(n = 1, rep(4,  species) )
  x.df <- as.data.frame(x)
  names(x.df) <- paste0("sp.", 1:species)
  return(x.df)
  
  } ) %>%
  bind_rows(., .id = "id") %>%
  pivot_longer(cols = c("sp.1", "sp.2", "sp.3"),
               names_to = "species",
               values_to = "RYe") %>%
  ggplot(data = .,
         mapping = aes(x = species, y = RYe, group = id)) +
  geom_line(alpha = 0.3) +
  geom_hline(yintercept = 0, linetype = "dashed", colour = "black") +
  scale_x_discrete(expand = c(0.03, 0.03)) +

This can be useful because, often, we donât know the expected relative yields. This method allows us to still calculate the biodiversity effects whilst incorporating uncertainty in the RYe values.

If RYe_post = FALSE, then we need to provide RYe values directly into the RYe argument. For each species, we need one of these values and, overall, the values must sum to one.

Now that we know how to use the isbell_2018_sampler() function, we will use it to calculate the biodiversity effects on some example data assuming known and unknown RYe values.

First, we define a dataset (Isbell et al.Â 2018, Table 2) with two species at two times and two places.

df.test1 <- 
  data.frame(sample = rep(c(1:4), each = 2), 
                  time=c(1,1,1,1,2,2,2,2), 
                  place=c(1,1,2,2,1,1,2,2), 
                  species=c(1,2,1,2,1,2,1,2),
                  M=c(100,50,100,50,50,100,50,100), 
                  Y=c(100,0,100,0,0,100,0,100))

To start with, we assume equal RYe values for the two species (i.e.Â 0.5 and 0.5) and we use the isbell_2018_sampler() function to calculate the biodiversity effects. We add df_test1 to the data argument and c(0.5, 0.5) to the RYe argument because we set RYe_post = FALSE because we know the RYe values.

df.test1.out <- Isbell_2018_sampler(data = df.test1, 
                                    RYe_post = FALSE, 
                                    RYe = c(0.5, 0.5))

Letâs view the output of the function. First, we can view the effects proposed by Isbell et al.Â (2018, Ecology Letters).

print(df.test1.out$Beff)

In this example data, the net biodiversity effect is 100 and it is completely explained by temporal insurance effects. The reason for this is that two different species dominate at the times where they are highest functioning in monoculture.

Second, we view the sum of local complementarity and local selection effects as calculated per Loreau and Hector (2001):

print(df.test1.out$L.Beff)
##   L.Beff Value
## 1     LC     0
## 2     LS   100

Next, we assume that we donât know the RYe. Therefore, instead of a single biodiversity effect, we will get a distribution of each effect. Here, we set RYe_post as TRUE, N = 100 and alpha_par as 4.

# create a test dataset
df.test2 <-

The output we get now is a data.frame with a column for sample. This column corresponds to a sample from the Dirichlet distribution. Now, instead of simply viewing the data in a table, we can plot it as a histogram.

require(ggplot2)

ggplot(data = df.test2.out,
       mapping = aes(x = Value, fill = Beff)) +
  geom_histogram() +
  facet_wrap(~Beff, scales = "free") +

As you can see from this set of histograms, we now have a histogram of biodiversity effects. These reflect differences in the value depending on the expected relative yield. However, you can also see that four effects: NO, SI, TC and TI have flat histograms. This is because these specific biodiversity effects do not depend on the RYe value and so will always only have the same value.
